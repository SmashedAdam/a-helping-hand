	- ## 重要概念
		- ### 1. 全球化
		  collapsed:: true
			- **特徵**：
			  collapsed:: true
				- 人們的交往突破 地域、 時間 、 平台 的限制
				- 資金、資源和產品的流動增加， 人口和資訊的流動也大增
				- 人際和國際的交流增加
				- 出現跨國企業和國際組織（分政府和非政府）等跨世界的聯係網絡
				- #### 總結：地理限制不再重要 、 時空壓縮 、 時間/空間 概念不强
			- **全球化的發展**：
			  collapsed:: true
				- #### 經濟全球化：
				  collapsed:: true
					- 國際組織制定經濟法規，規範各國的經濟活動
					- 跨國企業透過國際分工建立跨國生產網絡，造成生產全球化
					- 國際貿易和電子商務迅速發展
					- 國際資金全球流動
					- 跨國企業組建全球銷售網絡
					- 本地勞工為國外企業付出勞力，人才跨境流動
					- #### 概念詞：
						- 國際分工
						- 金融市場一體化
						- 消費全球化
						- 生產全球化
				- #### 文化全球化：
				  collapsed:: true
					- 各地人民交流增加，主要體現在資訊和人際交流，商品的流動
					- 全球普遍價值觀(自由，平等，公義等)逐漸形成
					- 文化和地域的聯係減弱，文化傳播超越特定地域和時空
					- 文化**單一化**和文化**多樣化**並存，兩者互相牽引
					- 西方文化向外傳播，成爲全球文化
			- **推動全球化的因素**：
			  collapsed:: true
				- **交通運輸**進步，縮短時間和空間的距離(時空壓縮)
				- **通訊科技**進步，促進人與人之間的交流
				- **傳媒發展**促進文化交流，跨國企業的發展
				- **國際組織**成立，推動全球化的發展
				- **全球議題**出現使國際之間的合作增加
			- **觀點**：
			  collapsed:: true
				- 經濟全球化：
					- 機遇：
						- 為不同的國家**創造更多的財富**
						- 各國經濟互相**融合、依賴**
						- 不同國家可以**調整產業結構**
						- **產品流動性增加**， 有助**提升人民生活素質**
					- 挑戰：
						- 富國投資弱國，可能造成**勞工剝削**、**血汗工廠**
						- **貧富差距惡化**， 威脅全球**經濟穩定性**
						- **削弱**發展中國家對經濟事務的**控制權**
						- **生產全球化**使得**發展中國家**的污染問題**日益嚴重**
				- 文化全球化：
					- 機遇：
						- 推動**文化交流**
						- **文化交流**提升**生活水平**，加强**各國溝通**，促成**世界大同**的**地球村**
					- 挑戰：
						- 全球品味**劃一**，各國文化被**「西方化」**，是**强國對弱國的侵略**，是**文化帝國主義**
						- 西方文化滲透到不同國家，可能與**本土文化**產生**文化衝突**
						- 威脅**文化多樣性**發展
						- **文化單一化**令某些傳統文化被**邊緣化**，甚至**消亡**

			- **1. 何爲公共衛生**：
			  collapsed:: true
				- **1. 公共衛生觀念的轉變**：
				  collapsed:: true
					- 傳統衛生觀念：
						- 成因：
							- 疾病是上天對世人的懲罰
						- 預防/診治：
							- 看巫醫、到聖地浸泡聖水
					- 近世生物醫學模式：
						- 成因：
							- 受到病原體感染
						- 預防/診治：
							- 注射疫苗、服食藥物
					- 現代公共衛生觀念
						- 成因：
							- 病原體感染外，也與不良生活習慣，惡劣環境衛生有關
						- 預防/診治：
							- 培養良好衛生習慣
							- 建立健康生活習慣
							- 改善環境衛生
							- 完善醫療系統
				- **2.公共衛生的定義**：
				  collapsed:: true
					- 概念圖 ![image.png](../assets/image_1655474667502_0.png)
				- **3. 公共衛生的重要性**：
				  collapsed:: true
					- 1. 維持公衆健康
					  2. 保持社會生產力
					  3. 提升生活素質
				- **4. 公共衛生的指標**：
				  collapsed:: true
					- 1. 出生時的平均預期壽命
					  2. 發病率和死亡率
					  3. 嬰兒夭折率
					  4. 安全食水的供應
					  5. 醫護人員與人口比例
					  6. 醫療衛生開支佔GDP百分比
				- **5. 個人與政府在推動公共衛生方面的責任**：
				  collapsed:: true
					- 個人責任：
						- 1. 保持個人健康
						  2. 保持環境衛生
						  3. 培養正確態度
					- 政府責任：
						- 1. 制定合適政策
						  2. 投放足夠資源
						  3. 改善環境衛生
						  4. 提高民衆意識
						  5. 與各地政府合作
				- **6. 政府制定公共衛生政策時的考慮因素**：
				  collapsed:: true
					- 1. 迫切性
					  2. 經濟效益
					  3. 公平原則
					  4. 可行性
					  5. 政策成效(針對性/起效速度/廣泛性/持續性)
			- **2. 大規模疫症蔓延事件與公共衛生發展**：
			  collapsed:: true
				- **1. 何爲傳染病？**
				  collapsed:: true
					- 傳染病三角模型
					- ![image.png](../assets/image_1655475665822_0.png)
					- 只要能消除其中一項，就能防止傳染病傳播
					-
					- 傳染病的主要傳播途徑
					- 1. 直接接觸
					  2. 空氣接觸
					  3. 病媒傳播
					  4. 體液傳播
					-
				- **2. 大規模疫症蔓延事件**：
					- 1. 嚴重急性呼吸系統綜合症
					  2. 禽流感
					  3. 後天免疫力缺乏症
					  4. 流行性感冒
				- **3. 影響疫症蔓延的因素**：
					- 1. 個人因素：衛生知識水平
					  2. 社會因素：社區環境衛生
					  3. 社會因素：跨國交流頻繁
					  4. 科學與科技因素：醫療科技發展
				- **4. 大規模疫症爆發的影響**：
					- 1. 損害人類健康
					  2. 加重財政負擔
					  3. 造成經濟損失
					  4. 削弱管治威信(大家認同政府的程度)
				- **5. 對大規模疫症事件的回應
					- 1. 個人方面
						- 1. 改善衛生習慣
						  2. 保持警覺性
					- 2. 政府方面
						- 1. 加强傳染病防控
						  2. 加强傳染病監察
						  3. 增加衛生資源投入
						  4. 加强教育和宣傳
						  5. 立法和加强執法
					- 3. 國際方面
						- 1. 加强醫療合作
						  2. 完善通報機制

			- **1. 何謂健康？**
			  collapsed:: true
				- **1. 健康的定義**
				  collapsed:: true
					- 生理健康：
						- 身體各個器官運作良好
					- 心理健康：
						- 有能力分辨和表達情緒
						- 用能力應付日常壓力，擔憂及情緒低落
					- 社交健康：
						- 能和他人建立和維繫良好關係
						- 能有效溝通
					- 健康狀態：
						- 健康：
							- 個人對生、心理、社會、文化的需求得到平衡，享受生命
						- 亞健康：
							- 出現生理和心理等方面的不適
						- 患病：
							- 生、心理有明顯的患病跡象
				- **2. 影響健康的因素**
				  collapsed:: true
					- 遺傳
					- 工作環境
					- 壓力
					- 衛生服務
					- 環境衛生
					- 住屋
					- 教育和經濟
					- 等......

			- **2. 生活模式與健康的關係**：
				- **1. 何爲生活模式病？**
				  collapsed:: true
					- 定義：
						- 與現代**城市**人的生活習慣有**密切關係**
						- 主要指**心臟病**、**中風**、**癌症**、**慢性呼吸道疾病**、**肥胖症**、**糖尿病** 等
						- 多屬**慢性病**，且**難以治愈**
						- 會令患者身體逐漸變差，引發各種并發症，損害身體健康
						-
				- **2. 引發生活模式病的原因**：
				  collapsed:: true
					- 1. 不良飲食習慣
					  2. 缺乏體能運動
					  3. 沉重生活壓力
					  4. 染上不良嗜好
				- **3. 生活模式病引申的影響**：
				  collapsed:: true
					- 損害個人健康
					- 加重醫療壓力
					- 增加財政開支
					- 阻礙國家發展
				- **4. 生活模式病的概況和預防**：
				  collapsed:: true
					- 癌症
					  collapsed:: true
						- 是指細胞變異，失控增生
						- 會擴散到其他器官，破壞身體機能，導致死亡
						- 吸煙、酗酒、不良飲食、缺乏運動、環境污染、接觸致癌物 都會提高患癌的風險
					- 對癌症的回應
					  collapsed:: true
						- 個人
							- 實踐健康生活
							- 定期進行身體檢查
						- 政府
							- 加强教育宣傳
							- 推廣健康計劃
							- 加强控煙政策
							- 改善空氣素質
						- 國際
							- 研究新的藥物
							- 研究癌症起因
							- 指定預防政策
							- 為發展中國家提供技術援助
						-
						-
					- 肥胖症
					  collapsed:: true

						- 增加患上**非傳染病**的風險
					- 對**肥胖症**的回應
